# Colored_Spinning_Donut
This is a colored spinning donut, fully simulated with just ASCII codes and implemented using C language. 


### Project Description
This is a colored spinning donut, fully simulated with just ASCII codes and implemented using C language.
      
### How To Build ?

To build the program: 

	1) gcc donut.c -o <execfilename>
	2) ./<execfilename>
